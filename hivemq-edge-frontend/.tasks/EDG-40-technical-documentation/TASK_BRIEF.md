# Task Brief: Technical Documentation

**Task ID:** EDG-40
**Task Name:** technical-documentation
**Date Created:** 2026-02-13

## Objective

Write comprehensive documentation summarizing the HiveMQ Edge Frontend technical stack from a developer's perspective.

## Requirements

Document the following areas:

1. **Core Toolchain**

   - Build system (Vite)
   - Language & runtime (TypeScript, Node.js)
   - Package manager (pnpm)

2. **Main Dependencies**

   - UI frameworks and components
   - State management
   - Data fetching
   - Routing
   - Forms and validation
   - Visualization libraries
   - Testing frameworks
   - Development tools

3. **Build, Testing, and Deployment Scripts**
   - Development scripts
   - Build processes
   - Linting and formatting
   - Testing (Cypress, Vitest)
   - Coverage reports
   - Deployment workflows

## Deliverables

- Suitable documentation structured in the repository

## Context

This documentation is for developers who need to understand:

- What technologies power the frontend
- How to build, test, and deploy
- Dependencies and their purposes
- Development workflow and tooling

## Success Criteria

- Organized by functional area (UI, testing, build, etc.)
- Clear explanation of each technology's role
- Suitable for new developers onboarding to the project
- Suitable for agents as a reference for technical details during development and troubleshooting
